What is this?

  1. A simple site to organize Wikipedia entries and related Youtube videos for topics of interest

  2. It also creates a few Matplotlib graphs and displays real time data showing counts of visited endpoints(routes) of this site.

How to Use

  1. Choose a library from the dropdown to look at the content or add new content

  2. Navigation bar:

    1. - Create a new library/database, then search/add the topics you want.

    2. - switch to a different library from the dropdown.

    3. - Enter search text, choose the number of pages, and then search Wikipedia. Choose from the resultant list and it will insert 20 related Youtube videos

    4. - return to this page

    5. - go to the graph creation/viewing page.

    6. - takes you to the 'site traffic' page.

  3. Library viewing/editing:

    1. You can add and delete entries by using the and icons.

    2. The view clicks are saved and will show up in the graphs.
