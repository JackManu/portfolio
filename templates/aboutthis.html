Wikipedia/Youtube Library

A Flask Python Web App designed using Visual Studio and published on PythonAnywhere.com

Feel free to add and delete entries if you want. Please do me a favor and click on at least one or two of the links to add to the view counts.

   How to use: Enter a search term, choose which Wikipedia pages you would like to save. The title of the Wikipedia page will be used as a search term to retrieve 50 links from Youtube.

  • RESTful APIs

    I set up API keys in the Wikipedia and Google API sites to enable connectivity.

    Wikipedia APIs requires an additional step to get an authentication token based on the API key (Curl command run through Python subprocess/check_output)

    The Python requests module is used to retrieve the json data from the sources

  • SQLite Database(3, count them, 3 WHOLE tables. lol. I've definitely dealt with more complex databases and SQL over the years.)
    •     Wikipedia Table

      Holds the necessary information for a given Wikipedia link.

    •     Youtube Table

      Same information as the Wikipedia table with the addition of a column 'wiki_id' to correlate to the Wikipedia entry

    •     view_counts Table

      Entries are inserted to this when the user clicks to view a Wikipedia or Youtube link. This seemed to be the easiest bit of data I could do graphing with

  • Data Analysis

    Currently showing a few different Python matplotlib graphs based on the data stored in this database.
